Personal Website

The website you're currently on! A personal project to learn more about web development. Built frpm scratch by hand in VSCode with HTML, CSS, and Javascript. Hosted with Github Pages.

Visneti

Build as a collaborative team over 3 months, Visneti is a mobile focused website developed for exploring local night-out venues in Liverpool. It uses a client-server architecture, with an HTML/CSS frontend and a PHP/SQL backend.

Wheel of Fortune

A Wheel of Fortune-style iOS app, built natively in Swift using Xcode. With responsive gameplay for word and phrase guessing, it features a scoreboard maintained in persistent memory and GUI design via Storyboard.

Submarine Chase

Submarine Chase is a JavaScript-powered browser game where players navigate their submarines to evade AI enemies. The grid-based gameplay uses HTML and CSS for an interactive user interface.
